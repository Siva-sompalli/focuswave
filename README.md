# FocusWave - Personalized Focus Optimization

## 📌 Overview
FocusWave is a machine learning-based web application that analyzes user productivity patterns and suggests optimal focus cycles.

## 🚀 Features
- Productivity tracking
- Distraction detection
- Focus prediction using ML
- Personalized recommendations

## 🛠 Tech Stack
Frontend:
- React.js
- Vite

Backend:
- Django
- Django REST Framework

ML:
- Pandas
- NumPy
- Scikit-learn

## ▶️ How to Run

### Frontend
npm install  
npm run dev  

### Backend
pip install -r requirements.txt  
python manage.py runserver  

## 📂 Project Structure
- frontend → React app  
- backend → Django API
